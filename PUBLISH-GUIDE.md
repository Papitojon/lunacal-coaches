# GitHub Pages Publish Guide

This site is ready to publish for free using GitHub Pages.

## What you need
- A free GitHub account
- Internet access
- About 5-10 minutes

## Publish steps
1. Go to GitHub and create a new repository.
   - Suggested name: `lunacal-coaches`
2. Upload all files from this folder to the repository root.
3. Make sure these files are in the root:
   - `index.html`
   - `compare.html`
   - `start.html`
   - `style.css`
   - `robots.txt`
   - `sitemap.xml`
   - `.nojekyll`
4. In the repository, open **Settings**.
5. Click **Pages**.
6. Under Source, choose **Deploy from a branch**.
7. Select:
   - Branch: `main`
   - Folder: `/ (root)`
8. Save.
9. Wait for GitHub to show your live URL.

## After it goes live
Once the live URL is available, update these files:
- `robots.txt`
- `sitemap.xml`

Replace `https://YOUR-LIVE-SITE-URL/` with the real GitHub Pages URL.

## Recommended first test
Open the live site and make sure:
- Home page loads
- Compare page loads
- Start page loads
- Affiliate CTA works
