# Lunacal for Coaches - Free Hosting Site

This is a simple static affiliate site that can be hosted for free using GitHub Pages.

## Files
- `index.html` - main landing page
- `compare.html` - Lunacal vs Calendly comparison
- `start.html` - direct CTA page
- `style.css` - shared styling

## Free hosting option: GitHub Pages
1. Create a free GitHub account if needed.
2. Create a new repository, for example `lunacal-coaches`.
3. Upload these files to the root of the repository.
4. In the repository settings, go to **Pages**.
5. Set source to **Deploy from a branch**.
6. Choose the `main` branch and `/root` folder.
7. Save.
8. GitHub will give you a free public URL.

## Suggested structure
